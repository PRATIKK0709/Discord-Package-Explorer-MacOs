# DiscSwiftPckg (WIP)

A SwiftUI application to visualize and explore your Discord Data Package.

## Features
- **Dashboard Overview**: Summary of your Discord activity.
- **Message Stats**: Analysis of messages by hour, day, and year.
- **Top Lists**: Top words, emojis, servers, and DMs.
- **Activity Charts**: Visual graphs of your activity patterns.
- **Avatar History**: View your past profile pictures.
- **Smooth Charts**: Beautiful visualization of usage trends.
